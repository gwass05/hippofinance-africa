A modern, interactive portfolio website for HippoFinance.africa, a registered microfinance institution offering digital credit solutions, device financing for Starlink, and SME lending across Africa.

## üöÄ Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd hippofinance-portfolio
```

2. Install dependencies:
```bash
npm install
# or
yarn install
```

3. Run the development server:
```bash
npm run dev
# or
yarn dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## üì¶ Build & Deploy

### Build for Production

```bash
npm run build
# or
yarn build
```

### Deploy to Vercel

1. Push your code to GitHub
2. Connect your repository to Vercel
3. Deploy automatically on push

Or deploy manually:

```bash
npm run build
vercel --prod
```
